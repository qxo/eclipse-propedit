【インストール手順】

  1. Eclipseの画面から「ヘルプ」→「ソフトウェアの更新」→「更新マネージャ」と選択して下さい。更新マネージャが開きます。

  2. 更新マネージャの左下の「フィーチャーの更新」ビューにおいて、「アクセス先サイト」を右クリックし、「新規」→「サイト・ブックマーク」で、サイト・ブックマークを作成してください。
    ・作成するサイト・ブックマークは下記の「URL」を入力し、「終了」ボタンを押してください。
    名前： 任意入力
    URL ： http://propedit.sourceforge.jp/eclipse/updates/
    ブックマークタイプ： Eclipse更新サイト

  3. サイト・ブックマークを作成すると、「フィーチャーの更新」の一番下に作成したブックマークが現れます。
    「jp.gr.java_conf.ussiy.app.propedit.eclipse.feature.PropertiesEditorFeature x.x.x」をクリックすると、右ウィンドウにプレビューが表示されます。右下あたりに、「すぐにインストール」ボタンがありますのでクリックしてください。

  4. インストールウィザードが開始しますので、そのままデフォルトで「次へ」ボタンをどんどんクリックしていってください。

  5. 「変更内容を有効にするには、ワークベンチを再起動する必要があります。すぐに再起動しますか？」というダイアログが表示されればインストール成功です。ダイアログにしたがって、Eclipseを再起動してください。
